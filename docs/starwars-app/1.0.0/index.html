SWAPI Starship demo

Sample console application using SWAPI for computing the numbers of stops the starships in a fleet must do to cover a distance given by the user in the command line.

Getting started

Prerequisites

You are going to need:

  • Linux, OS X or Windows
  • Git version control
  • NodeJS runtime environment, version 8.2.1 or newer
  • NPM package manager, 5.3.0 or newer

Installing

Reproduce the following steps to get the project up and running:

Clone this repository.

$ git clone https://github.com/JaxonDvl/starwars-app

Install dependencies in the source directory.

$ cd starwars-app
$ npm install

If we are going to deploy this app to a production environment we may want to omit development dependencies for testing and docs generation. Just install the dependencies with the --production flag:

$ cd starwars-app
$ npm --production install

Usage

Make sure you are in the starwars-app directory. You can configure the environment variables in .env file.

STARWARS_SHIPS_URL=https://swapi.co/api/starships
FLEET_FILE=shipfleet.json

Run the app using:

$ npm start

Testing

Docs

Install JSDoc 3 documentation generator for JavaScript:

$ npm install -g jsdoc

If you installed the app in production mode as seen in Installing chapter, make sure you install the node modules without the --production flag to include Docdash template module in our source tree:

$ cd starwars-app
$ npm install

Generating the docs: 

$ cd starwars-app
$ npm run docs

The output will be available in the docs folder and for this reporsitory on GitHub Pages at Star Wars App Docs

For future development we can include or exclude files that we want to document in jsdoc_conf.json file:

"source": {
        "include": ["db","helpers","models", "consoleApp.js"],
        "includePattern": ".+\\.js(doc|x)?$",
        "excludePattern": "(^|\\/|\\\\)_"
    }
